# Syntax.js - Options:

Below are all the options that can be passed to the "setOptions()" public function.


### Standard Options:
<br/>

| Type: | Name: | Description: |
| --- | --- | --- |
| *boolean* | showCopyButton | States if the copy buttons should be added (defaults to true). |
<br/>


### Translatable String Options:
<br/>

| Type: | Name: | Description: |
| --- | --- | --- |
| *string* | copyButtonText | The text that should be displayed for the "Copy" button. |
